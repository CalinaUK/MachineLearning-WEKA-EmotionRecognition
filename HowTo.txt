// How to activate
On macOS and Linux:
source venv/bin/activate

On Windows:
.\venv\Scripts\activate

// Check to it's activated
On macOS and Linux:
which python
should return: ".../venv/bin/python"

On Windows:
where python
should return: ".../venv/bin/python.exe"

// How to deactivate
deactivate
